SIGA

Plataforma de Governança Operacional construída sobre Microsoft Power Platform

Arquitetura

Core Estrutural (Dataverse)
• Security Roles nativos
• Business Units e Times
• Tabelas estruturais protegidas

Camada Administrativa
• Model-Driven App
• Configuração controlada

Camada Operacional
• Canvas App leve
• Permissões cacheadas
• Sem EAV massivo

Modelo de Segurança

  • Segurança real baseada em Security Roles do Dataverse
  • Controle funcional complementar via SIGA_Permissoes
  • Nenhuma lógica de segurança apenas na interface
  • Isolamento completo por tenant da cooperativa

Escalabilidade

  • Modelo meta-driven híbrido
  • Evita estrutura EAV pura
  • Compatível com Power BI
  • Pronto para expansão multiunidade

Modelo de Implantação

  • Entrega via Solution Managed
  • Instalação no tenant da cooperativa
  • Sem dependência externa
  • Sem conectores não governados
